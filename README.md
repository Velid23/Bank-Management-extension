# Bank-Management-extension
This project is an advanced extension of the Bank Management System developed in C++, upgrading the core data platform into a fully functional transactional banking simulator. Beyond standard CRUD operations (Create, Read, Update, Delete) for clients, this extended system features a dedicated Transactions Menu capable of managing dynamic operations like monetary deposits, balance withdrawals with overdrawing validation, and tracking real-time aggregated funds across all accounts via a flat-file repository (Clients.txt). The implementation serves as a vital practical step for mastering mid-to-advanced software concepts, emphasizing complex loop controls, conditional error checking, menu layout routing using structured Enums, custom tokenized string splitting algorithms, and vector-backed dynamic data manipulation. This source file retains both my custom implementation logic and the commented architecture blocks inspired by my instructor, Prof. Abou-Hadhoud. Incorporating both versions directly within this single environment served as an invaluable educational exercise, allowing me to contrast data management styles, master transactional logical flows, evaluate code modularity, and learn from my mistakes to write robust, maintainable, and highly optimized code.
